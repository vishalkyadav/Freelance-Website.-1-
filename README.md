# The Doctor Gadget • Vishesh Mobiles

Modern responsive business website for a mobile repair and smartphone sales shop with two branches.

## Tech Stack

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Poppins font (Google Fonts)

## Quick Start

1. Open `index.html` in your browser, or use a local server:
   ```bash
   npx serve .
   ```
2. Replace placeholder contact details (see below).

## Customize Before Launch

### Phone & WhatsApp
Search for `918770050980` in `index.html` and `js/main.js` and replace with your real numbers.

### Formspree
1. Sign up at [formspree.io](https://formspree.io)
2. Create a form and copy your form ID
3. In `index.html`, replace `YOUR_FORM_ID` in the contact form `action`:
   ```html
   action="https://formspree.io/f/YOUR_FORM_ID"
   ```

### Shop Addresses & Maps
Update addresses and Google Maps embed URLs in the **Locations** section of `index.html`. Get embed code from Google Maps → Share → Embed a map.

### Gallery Images
Replace Unsplash URLs in `js/main.js` (`galleryImages`) with your own shop photos.

### Social Links
Update footer social media `href` attributes.
